# Anti-Text

## Description
Anti-Text IoT device built using Intel Edison, Autodesk Fusion 360, Relayr.io Wunderbar, Seeed Grove-Starter Kit Plus (button, buzzer, rgblcd,
led).

## Setup Instructions
Git clone the repository. Run the site using:
```
nodejs server.js
```

And in the browser:
```
localhost:8888
```
